# AGENTS.md — 风暴中的大顺朝 项目地图

## 项目简介

一款基于 Ren'Py 引擎的架空历史悬疑悲剧视觉小说游戏，以架空大顺王朝《大顺律》兴衰史为核心，通过四段多视角碎片化叙事展现信息差悲剧。本次交付序章「垂暮」，含1条主线剧情+2个分支结局。

## 开发约束

1. **引擎锁定**：仅使用 Ren'Py 最新稳定版原生功能，不引入非必要第三方插件
2. **目录规范**：代码逻辑文件存放于 `game/`，剧情脚本存放于 `game/story/`，图片存放于 `game/images/`，音频存放于 `game/audio/`
3. **风格锁定**：水墨国风、褪色史书质感、无清晰人脸、靠剪影/姿态/光影叙事，禁止偏离
4. **内容锁定**：架空历史、严肃正史向、无玄幻/超自然/穿越，禁止私设人设、禁止增删剧情
5. **交付范围**：序章「垂暮」= 1条四段式主线 + 结尾1处双选项分支，不超出此范围
6. **素材格式**：PNG 立绘/背景、MP3 背景音乐、OGG 音效
7. **代码规范**：遵循 Ren'Py 官方脚本语法，Python 辅助逻辑仅用于分支判断与状态管理
