# AGENTS.md

## 项目核心简介

本项目为架空历史悬疑水墨风视觉小说序章，以大顺王朝《大顺律》十二年兴衰为核心叙事，四段多视角碎片化剧情，单分支双结局，主打历史厚重感与信息差悲剧，网页端可直接游玩。

## 技术栈说明

- 核心引擎：Ren'Py 最新稳定版
- 脚本语法：标准 Ren'Py 脚本 + 少量 Python 辅助逻辑
- 编译工具：renpyweb，将项目打包为纯静态网页
- 素材格式：WebP 图片、MP3 音频，全兼容网页端
- 部署平台：GitHub 仓库托管 + GitHub Pages 公开访问
- 存储方案：浏览器 LocalStorage 实现网页存档

## 强制开发约束

1. 新版强制最高优先级约束：仅允许替换本次更新的全套剧情设定与李东升临终台词，所有美术、立绘、色彩、文字样式、UI、动效、工程代码、部署流程、目录结构、素材命名规则完全禁止修改。
2. 世界观新版剧情严格遵循本次锁定文档，禁止私设历史节点、新增角色、修改人设。
3. 绝对禁止自行生成任何图片、背景、立绘素材，禁止代码绘图、矢量色块占位、纯色背景生成行为。
4. 所有背景图、角色立绘、UI底图、特写素材统一由独立文生图AI产出；本工程仅负责定义文件名、存放路径、编写Ren'Py素材调用代码。
5. 若素材未就位，仅在代码对应位置添加注释 `# 待美术素材就位：对应文件名xxx.webp`，禁止生成任何临时占位图形。
6. 美术严格对标《无悔华夏》低饱和水墨古卷质感，无写实照片、无3D渲染、无清晰人脸刻画。
7. 功能适配网页端运行，剔除PC端专属API、本地文件读写、系统弹窗等不兼容特性。
8. 所有素材命名、路径统一规范，保证素材放入后可直接编译运行，无路径报错。
