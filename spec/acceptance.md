# acceptance.md

## 剧情验收硬性标准

1. 李东升开口结局遗言必须严格为“老百姓以后会怎么样，只有天知道。”，禁止任何字词改动。
2. 完整新版十二年时间线、人物冲突、御书房对话、顾子明悬疑设定必须落地。
3. 默然离世结局中，李东升不得出现任何临终台词。
4. 统一收尾必须显示“百姓心中有杆秤 时间会证明一切”，停留3秒后淡出并返回主菜单。
5. 其余美术、功能、部署、性能验收标准全部沿用原版。

## 美术边界验收

1. Codex 不得生成任何背景图、立绘、UI底图、特写素材。
2. 素材未就位时不允许用占位图替代，也不允许用几何图、纯色块、代码绘图替代。
3. `game/images/background/` 与 `game/images/character/` 在外部素材导入前必须保持无图片文件。
4. `game/images/asset_list.md` 必须列出全部所需背景、立绘、群像与路径。
5. 剧情脚本中所有 `scene`、`show`、`image` 语句必须严格对应 `asset_list.md` 文件名。
6. 素材全部就位后，必须执行数量、命名、格式、尺寸、透明通道、路径调用验收。

## 当前环境验收限制

当前工作区未提供 Ren'Py/renpyweb 可执行工具和旧版工程，因此只能验证脚本文本、目录结构、素材清单、路径调用和静态网页代码。正式发布前仍需在原 Ren'Py 工程中进行编译与网页端运行验收。
