# 美术规格文档 — 风暴中的大顺朝

## 一、整体风格

低饱和度水墨国风、褪色古卷、史书质感、留白压抑、无清晰人脸、靠剪影/姿态/光影叙事。

## 二、固定UI色值体系

| 色彩角色 | 色值范围 | 用途 |
|----------|----------|------|
| 主色墨黑 | #2C2C2C – #1A1A1A | 文字、边框、背景底色 |
| 辅色绢黄旧纸 | #D4C5A9 – #C2B28A | 面板背景、文本框底色 |
| 点缀朱砂红 | #8B3A3A – #A0524F | 标题装饰、按钮高亮、印章 |
| 石青灰蓝 | #5B7A8C – #4A6A7A | 次要文字、图标、分割线 |
| 赭石土褐 | #8B6B4A – #6B4B3A | 装饰边框、纹理 |
| 留白米白 | #F5F0E8 | 高光区域、留白空间 |

## 三、角色呈现规则

1. **常态**：全身/半身剪影，靠服饰轮廓、姿态、道具区分身份，无五官
2. **情绪时刻**：以手部/局部特写替代表情（如握拳、颤抖的手、放下茶杯的手）
3. **宏大场景**：远景小点景人物，突出环境压抑感
4. **禁止**：任何角色出现清晰五官正面像

## 四、六大固定场景美术规范

### 场景1：皇城寝殿
- **色调**：暗冷调，深蓝灰底 + 微弱烛光暖点
- **元素**：龙床帷幔（暗色剪影）、烛台、空旷殿宇柱影
- **氛围**：权力中心的寂灭感，空间宏大但人物渺小

### 场景2：京城茶馆
- **色调**：暖调，绢黄底 + 朱砂点缀
- **元素**：茶桌、说书台、悬挂灯笼剪影、茶客剪影群
- **氛围**：市井烟火，但暗含历史沧桑

### 场景3：北方枯村
- **色调**：枯灰调，灰白底 + 赭石土褐
- **元素**：枯树、断壁残垣、风雪/尘沙、远山灰影
- **氛围**：绝望、荒芜、生命凋零

### 场景4：南方行辕
- **色调**：虚浮金绿调，暗绿底 + 金色装饰线
- **元素**：书案、卷轴堆叠、窗外虚华景色、烛光
- **氛围**：新秩序的虚浮繁华，理性到冷酷

### 场景5：朝堂
- **色调**：庄严暗调，墨黑底 + 朱砂柱
- **元素**：龙椅（空）、朝柱群、群臣剪影
- **氛围**：权力仪式的空壳化

### 场景6：西北回忆军营
- **色调**：苍凉黄褐调
- **元素**：帐篷、旗帜残影、沙场远影
- **氛围**：帝王回忆中的开国豪情，已成泡影

## 五、固定转场特效

| 转场名称 | 触发场景 | 效果描述 |
|----------|----------|----------|
| 墨滴晕开 | 回忆↔现实切换 | 黑色墨滴从中心扩散，晕染全屏后切换 |
| 卷轴展开 | 时间跳转 | 画面如卷轴从右向左展开，新场景显现 |
| 朱砂印章 | 抉择弹出 | 分支选项以朱砂印章形式盖在画面上 |
| 暗角幻觉 | 帝王幻觉 | 画面四角暗化收缩，中心模糊扭曲 |
| 毛笔逐字 | 标题显示 | 标题文字以毛笔书写效果逐字显现 |

## 六、字体规范

| 用途 | 字体风格 | 备注 |
|------|----------|------|
| 正文对话 | 仿宋 | 清晰可读，历史质感 |
| 内心独白 | 行书 | 区分对话与心理活动 |
| 章节标题 | 隶书碑刻 | 庄严厚重 |
| 选项文字 | 篆刻朱砂 | 朱砂红色，刻印质感 |
| 旁白叙述 | 楷书 | 正式、客观 |
