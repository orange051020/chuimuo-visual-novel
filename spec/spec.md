# 主规格文档 — 风暴中的大顺朝

## 项目目标

打造一款沉浸式架空历史悬疑悲剧视觉小说，以「信息不对等 + 立场对立 + 制度惯性」为悬疑内核，通过四段多视角碎片化叙事展现一场「全员无恶人、全员皆悲剧」的历史崩塌。

## 交付范围（序章：垂暮）

| 交付项 | 具体内容 |
|--------|----------|
| 主线剧情 | 4个固定叙事片段，线性推进 |
| 分支结局 | 仅结尾1处双选项分支（选项A：临终留憾独白 / 选项B：默然离世） |
| 剧情文本 | 3500–4500字完整中文叙事（不含代码） |
| 游戏UI | 主菜单、存档/读档、系统设置、文本回放（History） |
| 美术素材 | 6大场景背景 + 8角色剪影立绘 + UI元素 + 转场特效 |
| 音频框架 | BGM/音效引用框架（占位文件，结构完整） |

## 模块划分

### 模块一：剧情脚本（game/story/）
- chapter1_market.rpy — 京城市井：后世定论，说书人盖棺
- chapter2_palace.rpy — 皇城寝殿：帝王临终幻觉
- chapter3_north.rpy — 北方乡野：底层信徒殉道
- chapter4_south.rpy — 南方行辕：权臣复盘大局
- endings.rpy — 双结局分支

### 模块二：游戏系统（game/）
- script.rpy — 角色定义、游戏入口
- options.rpy — 游戏配置
- gui.rpy — GUI色值与字体配置
- screens.rpy — 全套界面定义

### 模块三：美术素材（game/images/）
- backgrounds/ — 6大场景：皇城寝殿、京城茶馆、北方枯村、南方行辕、朝堂、西北军营
- characters/ — 8角色剪影：李东升、刘思廉、马靖、顾子明、徐淑娘、李承英、韩铁山、赵铁栓
- ui/ — 标题画面、按钮、边框装饰

### 模块四：音频素材（game/audio/）
- bgm/ — 4段场景配乐占位（市井暖调、寝殿暗冷、枯灰调、虚浮金绿）
- sfx/ — 转场音效占位（墨滴、卷轴、印章、书写）

## 叙事结构

```
片段一：京城市井（暖调）
  ↓ 线性推进
片段二：皇城寝殿（暗冷调）
  ↓ 线性推进
片段三：北方乡野（枯灰调）
  ↓ 线性推进
片段四：南方行辕（虚浮金绿调）
  ↓ 分支选项
  ├→ 选项A：临终留憾独白 → 结局A
  └→ 选项B：默然离世     → 结局B
```

## 四重信息差悲剧结构（贯穿全篇）

1. 帝王深宫信息茧房，永远滞后于真实民间
2. 底层信徒至死等待帝王救赎，不知帝王早已无力回天
3. 权臣十年隐忍积累绝望，非篡位野心而是制度认知对立
4. 首辅顾子明提前洞悉全部惨剧，沉默隐忍一年，心理悬疑贯穿
