# 剧情规格文档 — 风暴中的大顺朝

## 一、架空大顺王朝世界观

### 1.1 建国背景
大顺建国彻底击溃明残余与后金主力。明朝残余逃亡东南亚建立「南明海廷」，仅沿海骚扰；后金主力覆灭、部落分裂，不再威胁北方。外部无患，大顺进入内部制度转型阵痛期。

### 1.2 李东升施政链条（开国1-6年）
1. 开国元年：分田赋、集体耕作、永久暂停科举
2. 开国2-3年：引入西洋机器、建设国营集体工场、设立火器局
3. 开国4-5年：改革失效，地方士绅官吏阳奉阴违，举荐制滋生腐败，旧秩序死灰复燃
4. 开国六年：绝望之下颁布《大顺律》，下放民告官、百姓审判官吏的至高权力

### 1.3 朝堂两条对立路线
- **皇权理想派（李东升）**：制度不破、剥削不止，必须自下而上赋予百姓权力
- **经济现实派（刘思廉、马靖）**：百姓需安稳饭碗而非权力，乱世源于激进变法，经济稳定优先一切

### 1.4 完整时间线
| 时段 | 事件 |
|------|------|
| 开国1-5年 | 建国改革、初见成效、暗流滋生 |
| 开国6-9年 | 大顺律试点推广、短期利民、朝野两极评价 |
| 开国10-14年 | 全面异化、体系变质、各方势力暗中结网、顾子明提前崩溃 |
| 开国15-16年 | 民变爆发、朝堂逼宫、大顺律废止、帝王停朝 |
| 开国16-18年 | 全国反攻倒算、底层支持者惨遭肃清、消息封锁、帝王至死不知情 |
| 开国19年 | 权臣掌权、通商开埠、正史彻底改写评价 |

## 二、核心人物人设（固定，禁止OOC）

### 李东升 — 理想主义开国帝王
- 身份：大顺开国皇帝
- 性格：孤高、疲惫、悔憾交织
- 核心矛盾：理想主义与现实崩塌的撕裂
- 呈现方式：临终幻觉中回望一生， Fragment式回忆

### 刘思廉 — 务实冷血经济能臣
- 身份：经济现实派领袖
- 性格：重秩序、重民生温饱、极度排斥激进变革
- 核心矛盾：十年隐忍积累绝望，非篡位野心而是制度认知对立
- 呈现方式：南方行辕冷静复盘，理性到冷酷

### 马靖 — 军方实权统帅
- 身份：军方领袖
- 性格：维稳优先，局势崩坏后武力站队
- 呈现方式：片段四中被提及，不直接出场

### 顾子明 — 律法起草者、核心悬疑人物
- 身份：首辅，《大顺律》起草人
- 性格：最早看破真相、隐忍崩溃
- 核心矛盾：提前洞悉全部惨剧，沉默隐忍一年
- 呈现方式：心理悬疑贯穿全篇，帝王回忆中的碎片

### 徐淑娘 — 皇后、监察女官体系创立者
- 身份：皇后
- 性格：初心清正，体系后期异化
- 呈现方式：帝王回忆中出现

### 李承英 — 帝王义子
- 身份：帝王义子
- 性格：初心赤诚，被地方士族腐蚀异化
- 呈现方式：帝王回忆中的痛心片段

### 韩铁山 — 北境武将
- 身份：北境守将
- 性格：借律法清洗异己、私固兵权
- 呈现方式：片段三中作为清算者的阴影

### 赵铁栓 — 退伍老兵、底层律法宣讲者
- 身份：退伍老兵，《大顺律》民间宣讲者
- 性格：纯粹殉道者，至死不渝的信仰
- 呈现方式：片段三的主角，殉道场景

## 三、序章剧情结构（固定四段式）

### 片段一：京城市井 — 后世定论
- **场景**：京城茶馆，多年后
- **视角**：说书人（旁观者/后世定论者）
- **色调**：暖调（市井烟火）
- **内容**：说书人醒木一拍，讲述大顺律兴衰，盖棺定论"祸国之法"。茶客议论，说书人点到即止，留下"真相谁说得清"的悬念。
- **叙事功能**：建立历史框架，暗示信息差主题
- **字数**：约900字

### 片段二：皇城寝殿 — 帝王临终
- **场景**：皇城寝殿，帝王弥留之际
- **视角**：李东升（第一人称幻觉/意识流）
- **色调**：暗冷调（权力中心的寂灭）
- **内容**：帝王临终幻觉，碎片式回忆开国改革、四人集团（刘思廉、马靖、顾子明、徐淑娘）的崩塌、义子李承英的异化。顾子明的沉默贯穿。
- **叙事功能**：展现帝王视角的信息茧房与悔憾
- **字数**：约1100字

### 片段三：北方乡野 — 底层殉道
- **场景**：北方枯村，清算现场
- **视角**：赵铁栓（第三人称）+ 旁白
- **色调**：枯灰调（绝望、荒芜）
- **内容**：赵铁栓被韩铁山部下拘押清算。他至死相信帝王会来救他，不知帝王早已无力。消息被封锁，深宫永远不知。殉道者的纯粹与制度的残酷形成核心悲剧。
- **叙事功能**：信息差悲剧的实体化呈现
- **字数**：约900字

### 片段四：南方行辕 — 权臣复盘
- **场景**：南方行辕，权臣书房
- **视角**：刘思廉（第三人称限制视角）
- **色调**：虚浮金绿调（新秩序的虚华）
- **内容**：刘思廉复盘大局，冷静分析大顺律的必然失败。非恶人独白，而是制度认知对立的理性陈述。新秩序成型，通商开埠在即。末尾引出帝王结局的分支选项。
- **叙事功能**：权臣视角补全历史全貌，引出分支
- **字数**：约900字

### 分支结局

#### 结局A：临终留憾独白
- **触发**：选项A
- **内容**：帝王临终前的最后独白，表达对改革失败的悔憾、对殉道者的愧疚、对制度惯性的绝望。约300字。
- **结束**：帝王气绝，画面渐暗，墨滴晕开

#### 结局B：默然离世
- **触发**：选项B
- **内容**：帝王无言，拒绝最后的独白。沉默中离世，殿外钟声响起。约200字。
- **结束**：画面定格于空荡寝殿，卷轴缓缓合上

## 四、叙事规则

1. **碎片化真相**：每个片段只呈现部分真相，完整图景需四段拼合
2. **信息滞后**：角色所知信息严格受限于其位置和时代，禁止"上帝视角"穿帮
3. **多视角补全**：说书人(后世)→帝王(权力中心)→百姓(底层)→权臣(实用派)，四视角构成完整历史
4. **留白叙事**：关键悲剧靠暗示和留白，不直绘暴力血腥
5. **无纯粹反派**：每个角色的行为都有其立场逻辑，悲剧源于制度而非个人
